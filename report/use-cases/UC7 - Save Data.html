UC7 - Save Data
Data:
  • <Taxpayer TRN>
  • <File Type>
Main Flow:
  1. User of the system issues `Save Data` command.
  2. User selects the type of format to export to.
  3. System uses the TRN provided to create the corresponding log file with given export type.
  4. System saves data in log file.
Wrong File Format
  1. User has provided a file type that is not supported.
  2. User is notified with: `Wrong file format.`.
  3. Use case ends.
IOException
  1. Occurs when a filesystem error happens mid opening log file for write.
  2. Use case ends.
