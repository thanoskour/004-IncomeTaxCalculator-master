UC6 - View Report
Data:
  • <Taxpayer TRN>
Main Flow:
  1. User of the system issues `View Report` command.
  2. System aggregates tax information of current taxpayer (using provided TRN).
  3. System calculates basic and total tax numbers, as well as the variation increase or decrease.
  4. System gets information about the receipts amount according to receipt types.
  5. System displays a pie chart and bar chart with current expendature types and tax analytics.
