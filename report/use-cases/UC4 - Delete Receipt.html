UC4 - Delete Receipt
Data:
  • <Receipt ID>
  • <Taxpayer TRN>
Main Flow:
  1. User of the system issues `Delete Receipt` command.
  2. User is prompted to provide a receipt id to delete.
  3. System deletes receipt with corresponding id from memory of the current taxpayer (using TRN).
  4. System executes UC5.
Invalid Receipt ID
  1. User has provided a receipt id that is invalid or one that does not exist in the list.
  2. Use case ends.
