UC8 - Delete Taxpayer
Data:
  • <Taxpayer TRN>
Main Flow:
  1. User of the system issues `Delete Taxpayer` command.
  2. User is prompted for a taxpayer TRN for deletion.
  3. System deletes taxpayer with corresponding TRN from memory.
No Taxpayer Loaded
  1. User tries to remove a taxpayer however no taxpayer is loaded yet.
  2. User is notified with: `There isnt't any taxpayer loaded. Please load one first.`.
  3. Use case ends.
Invalid Taxpayer TRN
  1. User has provided a TRN that is not valid (per requirements of the system).
  2. User is notified with: `Adress number is not a valid number.`.
  3. Use case ends.
